Download Help

Data for one or more sites may be downloaded for analysis.

The data download is available from the "Site Identification" tool, for download of data from one site, or from the "Multi-Site Download Tool", for data from multiple sites.

Single-Site Download

At present, data for a single site is available as an XLS (Microsoft ® Excel ® format) file, with three sheets:

  • Well Log
  • Water Levels
  • Water Quality

The Well Log sheet contains both Lithology and Construction information.

The format for the single-site download will soon change to be identical to that for the Multi-Site Download (q.v.).

Multi-Site Download

The multi-site download tool provides data for all the sites visible on the map, which can be controlled using the agency, aquifer, and network selection filters (in the left sidebar of the map) and the drag and zoom tools.

The download tool allows you to choose which data categories to download for the visible sites, choosing from:

  • Water Level
  • Water Quality
  • Construction
  • Lithology

The data is downloaded as a zip file, containing one CSV (comma-separated values) file for each selected data category; the columns in each CSV file are labeled with column headers. The columns are separated by the comma character (ASCII 44), data is delimited by the quote character (ASCII 34), and each row of data is terminated by a linefeed character (ASCII 10).

The zip file also contains a REGISTRY.csv file, giving general information about all the visible sites.
